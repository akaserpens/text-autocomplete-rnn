# Задача проекта
* Есть входной датасет с короткими текстовыми постами
* Взять датасет, очистить его, подготовить для обучения модели.
* Реализовать и обучить модель для предсказания продолжения фразы на основе рекуррентных нейронных сетей.
* Замерить качество разработанной и обученной модели.
* Взять предобученную модель из Transformers и замерить её качество.
* Проанализировать результаты и дать рекомендации: стоит ли использовать лёгкую модель или лучше постараться поработать с ограничениями по памяти и использовать большую предобученную.

# Структура проекта
* /configs - содержит файлы конфигурации
* /data - директория для хранения исходного и промежуточных датасетов
  * tweets.txt - исходный файл с данными
  * tweets_preprocessed.csv - данные после очистки
  * tweets_train.csv - train выборка
  * tweets_val.csv - validation выборка
  * tweets_test.csv - test выборка
* /models - содержит веса обученных моделей
* /src - исходный код
